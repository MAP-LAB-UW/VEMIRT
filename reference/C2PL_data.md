# Simulated Data Set for Confirmatory M2PL Analysis

Responses are simulated based on an M2PL model with 2 factors. The true
factor correlations are set as 0.8.

## Usage

``` r
C2PL_data
```

## Format

A list of components of the data set:

|          |                                                        |
|----------|--------------------------------------------------------|
| ` data`  | Item responses                                         |
|          |                                                        |
| `model`  | Loading indicators                                     |
|          |                                                        |
| `params` | True parameters used for generating the item responses |
